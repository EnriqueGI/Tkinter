Introducción a Tkinter

Tkinter es la librería estándar de Python para crear interfaces gráficas de usuario (GUI). Es una herramienta potente y relativamente sencilla de aprender que viene incluida por defecto con la instalación de Python, permitiéndote construir aplicaciones de escritorio de forma rápida.
